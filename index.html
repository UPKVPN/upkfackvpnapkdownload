Application အသုံးပြုပုံ လမ်းညွှန်

UPK FAST VPN PRO ကိုဖုန်းထဲထည့်ပြီးသွားပါက အောက်ပါအဆင့်များအတိုင်း လွယ်ကူစွာ စတင်အသုံးပြုနိုင်ပါပြီ။

  1. ၁
    App ကိုဖွင့်ပါ

    သင်၏ဖုန်း Screen ပေါ်ရှိ UPK FAST VPN PRO icon ကိုနှိပ်၍ app ကိုဖွင့်ပါ။

    [ဒီနေရာတွင် App Icon ပုံ သို့မဟုတ် App ၏ ပင်မစာမျက်နှာပုံ ထည့်ရန်]

  2. ၂
    Server ရွေးချယ်ပါ

    App အတွင်းရောက်ပါက ဆာဗာရွေးရန်နေရာ (ဥပမာ - 'Select Server') ကိုနှိပ်ပြီး သင်အသုံးပြုလိုသောနိုင်ငံ (Singapore, Thailand) ကိုရွေးချယ်ပါ။
    (သိမှတ်ရန်- Singapore server သည် အာရှတွင်း အမြန်ဆုံးဖြစ်လေ့ရှိသည်။)

    [ဒီနေရာတွင် Server List ပြထားသည့်ပုံ ထည့်ရန်]

  3. ၃
    ချိတ်ဆက်ပါ (Connect)

    ဆာဗာရွေးပြီးပါက Connect Button (အလယ်ရှိ အဖွင့်/အပိတ် ခလုတ်ကြီး) ကိုနှိပ်ပါ။ ပထမဆုံးအကြိမ်ဖြစ်ပါက သင့်ဖုန်းမှ Connection Request တောင်းလျှင် 'OK' နှိပ်ပေးပါ။

    [ဒီနေရာတွင် Connect Button နှိပ်ရမည့်ပုံ ထည့်ရန်]

  4. ၄
    ချိတ်ဆက်မှု အခြေအနေစစ်ဆေးပါ

    Connect ဖြစ်သွားပါက ခလုတ်သည် 'Connected' ဟုပြပြီး အရောင်ပြောင်းသွားပါမည်။ ဖုန်း၏အပေါ်ဘက် Status Bar တွင် သော့ခလောက်ပုံစံ (VPN icon 鑰) ပေါ်လာပါမည်။

    [ဒီနေရာတွင် Connected ဖြစ်နေသည့်ပုံ ထည့်ရန်]

  5. ၅
    အဆက်ဖြတ်တောက်ရန် (Disconnect)

    အသုံးမပြုတော့ပါက App ကိုပြန်ဖွင့်ပြီး 'Disconnect' ဟုပြနေသော ခလုတ်ကို ပြန်နှိပ်လိုက်ရုံပါပဲ။

    [ဒီနေရာတွင် Disconnect လုပ်ရမည့်ပုံ ထည့်ရန်]
